# SKILL: DEBUG ERROR

## CONTEXT

You are working in a Next.js + Prisma + TypeScript environment.

---

## OBJECTIVE

Analyze an error and provide a precise solution.

---

## RULES

* Identify the ROOT cause of the error
* Explain WHY it happens
* Provide a concrete fix
* Be specific (no generic advice)

---

## ANALYSIS PROCESS

1. Read the error message
2. Inspect related code
3. Identify incorrect logic or configuration
4. Propose minimal fix

---

## RESTRICTIONS

* DO NOT give generic debugging advice
* DO NOT suggest unnecessary refactors
* DO NOT rewrite full code unless needed

---

## OUTPUT FORMAT

* Short explanation of the problem
* Exact fix (code or steps)

---

## USAGE

Use this skill when:

* Prisma errors
* API failures
* TypeScript issues

Example request:

"This endpoint throws an error: [paste error + code]"
